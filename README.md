图床
====

存放图片的地方。

## 引用 SVG 格式图片

在 Raw 格式的图片 URL 后添加 `?sanitize=true`，比如：

```
![Java 容器分类](https://github.com/Lazyb0x/Images/raw/master/svg/simple-container-taxonomy.svg?sanitize=true)
```

效果：

![Java 容器分类](https://github.com/Lazyb0x/Images/raw/master/svg/simple-container-taxonomy.svg?sanitize=true)
